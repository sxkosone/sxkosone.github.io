A look under the hood

I started teaching myself web development so that I could create functional and inspiring websites. An intensive, immersive Software Engineering bootcamp at Flatiron School gave me the skills to build web applications, from robust backends to interactive, performant frontends.

I am a full-stack web developer with experience in React and JavaScript on the frontend and Ruby on Rails and PostgreSQL on the backend. I have built RESTful APIs and used Redux for state management. I have also gained solid HTML5 and CSS skills, with libraries such as Semantic UI and Bootstrap.

Chatbot Builder

A web platform for chatbot creation. Users can script dialogues and use machine learning to train their chatbots to have conversations. Main Technologies: React, Redux, Machine learning

Meet by Me

Mapping application to show Meetup events near you. Users can search, filter and save events. Frontend uses Mapbox and React, Rails backend fetches the information from Meetup API. Main technologies: Ruby on Rails, Mapbox, React, JavaScript, Semantic UI

Robot Rap Generator

Add drums, a background beat and lyrics to create your own robotic rapsong. All front-end functionality built with vanilla JavaScript. RESTful backend API built with Ruby on Rails. Main technologies: Ruby on Rails, JavaScript, Bootstrap.

Job Search Platform

JOBuddy is a full Ruby on Rails app that enables users to sign up, browse a Stackoverflow job post feed from their city and initialize new applications. Main technologies: Ruby on Rails, REST, Bootstrap.

My First (this) Website

This is what went into this website. I started with the basics, then added extra features and made it responsive. Main technologies: HTML, CSS, Bootstrap.

A Grocery List App

I wanted a super simple app to contain my shopping list, add items and cross over found items. Just the basics, no extras. Main technologies: JavaScript, Vuejs.

A Game of Checkers

An ongoing pair programming project to create a game of checkers with a chat option for the players. Main technologies: HTML, Flexbox, CSS, JavaScript, DOM manipulation, Vuejs.
